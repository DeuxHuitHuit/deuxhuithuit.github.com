Deux Huit Huit's open source projects and labs

Project: jQuery Quick Search

A jQuery plugin for searching through tables, lists, etc quickly. We maintain this fork since the original author does not do it anymore.

Github page: http://deuxhuithuit.github.io/quicksearch/
Online demo: http://deuxhuithuit.github.io/quicksearch/r/examples/
http://deuxhuithuit.github.io/quicksearch/r/examples/super_table.html
jQuery plugin page: http://plugins.jquery.com/jquery.quicksearch/

Lab: Canvas Color Dodge Effet

Javascript color dodge with canvas with random displacement.

Github page: http://deuxhuithuit.github.io/color-dodge/
Online demo: http://deuxhuithuit.github.io/color-dodge/demo/

Project: jQuery Event Based Animation

jQuery plugin that offers a different way to do animation. End (or target values) can be changed during the animation.

Github page: http://deuxhuithuit.github.io/jQuery-event-based-animation/
Online demo: http://deuxhuithuit.github.io/jQuery-event-based-animation/r/tests/jquery.event-based-animation.js.adhoc-test.html
Unit tests: http://deuxhuithuit.github.io/jQuery-event-based-animation/r/tests/jquery.event-based-animation.js.test.html

Project: jQuery Sizing

Collection of function that performs specific sizing such as making the same size or resizing according to a certain ratio.

jQuery plugin page: http://plugins.jquery.com/jquery.sizing/
